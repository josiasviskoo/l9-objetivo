# Plugin WordPress — Objetivo

Plugin WordPress a ser desenvolvido a partir do layout de referência em [`../layout-apresentado/`](../layout-apresentado/) (não editar aquela pasta).

Ainda não implementado. Próximos passos ao iniciar o desenvolvimento:

1. Definir o slug do plugin (sugestão: `objetivo-l9`).
2. Criar o arquivo principal `objetivo-l9.php` com o cabeçalho padrão de plugin WordPress.
3. Portar seções do `index.html` (segmentos, timeline, blog) e do `shop.html` para templates/blocos do plugin.
4. Migrar imagens de `layout-apresentado/assets/img/` e `layout-apresentado/logos/` para `assets/` deste plugin.
